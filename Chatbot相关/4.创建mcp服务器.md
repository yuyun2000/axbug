# 首先创建demo服务器跑一边流程
参考流程：https://modelscope.cn/headlines/article/1193

- 创建python 3.10虚拟环境
- 安装uv（可能需要挂代理）
```
export HTTP_PROXY=http://ip:7890
export HTTPS_PROXY=http://ip:7890
```

```
curl -LsSf https://astral.sh/uv/install.sh | sh

echo 'export PATH="$HOME/.local/bin:$PATH"' >> ~/.bashrc

source ~/.bashrc
```
